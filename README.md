 # SENAI CIÊNCIA DE DADOS - AULA 28 - **DESAFIO TUDO QUE VIMOS ATÉ AQUI**

## Solução para o Desafio

1ª *Requisitos Funcionais* :

- Tratamento de dados das notas que chegam;
- Automatizar processo de calcular notas, aluno por aluno, e separar que está de recuperação;
- Destacar os que tiverem melhores notas de forma automática;
- Destacar alunos com piores rendimentos e assim a área pedagógica apoiar o aluno'
- O sistema deve ter a função principal do processamento de informações;
- Gerar um arquivo final de forma clara e com os dados processados. 


2ª *Requisitos Não Funcionais*:

- Linguagem em Python;
- Confiabilidade, em que os cálculos sempre executem de forma correta;
- Obrigatoriamente o sistema deve exportar resultado;
- Desempenho bom, em que os cálculos ocorram de forma rápida;
- Sistema deve prever, dados incompletos, corrompidos sem travar.
